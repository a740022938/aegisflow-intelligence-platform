# v6.3.0 范围锁定 — 自动回流 v1

## 版本定位
建立反馈回流闭环：失败样本、低置信度样本、人工标记样本沉淀到回流池，支持后续再训练与数据集迭代。

## 必须完成
1. feedback_batches + feedback_items 数据表
2. 8+ REST API（CRUD + 过滤 + 导出）
3. 最小 UI 页面
4. 3 类回流来源：failed_case / low_confidence / manual_flag
5. 导出候选数据集 manifest
6. 审计留痕
7. Windows 回退脚本
8. 收口报告与备份

## 不在范围
- 复杂可视化标注系统
- YOLO / OpenClaw 扩展
- 自动触发回流（仅手动登记）
- 数据增强 / 再训练执行
- Dark/Light 模式

## 绝对约束
- 不破坏 v6.0.0 / v6.1.0 / v6.2.0
- 可回退、可审计、可归档
