# AGI Model Factory v4.5.0 — Page Quality Pass & Vision Surface Expansion

## 1. 版本目标
本版本为页面质量通关版本，不新增模型模板、不重做后端主链。重点是统一页面信息层级、增强 Vision 主链露出、补强跨页面联动与可操作路径。

## 2. 范围锁定
- 包含：共享 UI 基建、Models/Deployments/Evaluations/Tasks 主改造，Approvals/Artifacts 次改造，Workflow Jobs 与 Datasets 的链路补强。
- 不包含：新模板扩展、Mahjong 专用流水线、后端重构、Dashboard 大改。

## 3. 页面级改造点
### 3.1 共享 UI 基建
- `shared.css` 增加统一结构类：`page-summary-strip`、`compact-stat-grid`、`detail-group`、`detail-kv-grid`、`inline-meta-list`、`destructive-action-row`、`linked-entity-chip`、`empty-state-actions`、`vision-surface-strip`、`entity-lineage-block`。
- `PageHeader` 支持 `meta` 与 `summaryStrip`。
- `SectionCard` 支持 `subtitle/description/meta/footer`，形成“标题-描述-主体-尾部”层级。
- `StatusBadge` 支持 tone/size 与空态文案。
- `DetailPanel` 支持 `subtitle/meta/footer`。
- `EmptyState` 支持标题、说明、主操作、次操作。

### 3.2 Models
- 新增头部摘要：total / released / draft / vision-capable / with artifacts。
- 详情区新增分组：Identity、Source/Lineage、Package/Export、Artifact/Report、Vision Surface、Related Evaluations/Experiments。
- 增加跨页入口：Models -> Training/Tasks/Workflow Jobs/Evaluations/Artifacts/Deployments。
- 删除操作降级为次级按钮，减少主视觉抢占。

### 3.3 Deployments
- 新增部署中心头部摘要：total deployments、active/idle/failed、last rollout、linked models。
- 列表补充来源与最近事件感知。
- 链路分组增强：Linked Model、Package/Artifact、Runtime Environment、Evaluation。
- 空态加入主次操作（新建 / From Artifact）。

### 3.4 Evaluations
- 头部加入 report-ready / linked datasets / compare entry 摘要。
- 列表项增强 metrics subline + model/dataset/time 信息。
- 详情区分组增强：Summary、Metrics、Dataset/Model Linkage、Report/Compare Entry、Artifacts/Exports。
- 显著增加 compare 入口（导向 models 对比入口）。

### 3.5 Tasks
- 头部摘要加入 selected workflow 感知。
- 详情增强：Task Summary、Current Execution、Pipeline Links、Vision Surface、Errors/Recovery、Related Outputs。
- 明确展示当前步骤、错误摘要、下一步入口、workflow/model/dataset/artifact 链路。
- 加强 Task -> Workflow Job 的可点击入口。

### 3.6 Approvals（第二优先级）
- 头部摘要加入 pending/approved/rejected/high risk。
- 详情增强 Risk & Impact，明确审批影响与风险级别。
- 加入来源入口跳转（Workflow Jobs / Tasks / Models）。

### 3.7 Artifacts（第二优先级）
- 头部摘要加入 ready/reports/linked objects。
- 强化 Source 与关联对象可读性与跳转（Tasks / Training / Evaluations / Deployments）。
- 细化空态与占位文案，降低“空壳页”感。

## 4. Vision Surface 扩展说明
- 统一实现 `VisionSurfaceStrip`，固定六段：`detect` / `handoff` / `segment` / `verify` / `track` / `rules`。
- 每段包含：名称、状态、摘要、可跳转关联对象。
- “有数据展示绑定信息、无数据显示未接入/未产出”已统一。
- 至少在 3 个页面稳定可见：
  - Tasks
  - Workflow Jobs
  - Models

## 5. Cross-page Linking 说明
已补强核心链路：
- Task -> Workflow Job（Tasks 详情 chips + Overview）
- Workflow Job -> Model / Dataset / Artifact（Workflow Jobs Linked Entities）
- Model -> Experiments / Evaluations / Reports（Models 分组与 tab 内链接）
- Dataset -> Source Task / Pipeline（Datasets source_task + pipeline 入口）
