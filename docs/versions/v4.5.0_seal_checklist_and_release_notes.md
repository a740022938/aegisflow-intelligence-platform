## 本轮必要修复
### Models 详情数据结构映射修复
- 问题：`/api/models/:id/detail` 返回结构为 `{ model, ... }`，页面按旧结构读取顶层字段，导致详情标题与多字段为空。
- 修复：在 `Models.tsx` 中将接口结果映射为页面使用结构（`model` 合并到 `selectedModel`，并保留 metrics/report/vision 相关字段）。
- 影响：恢复 Models 详情可读性与详情跳转可用性。
