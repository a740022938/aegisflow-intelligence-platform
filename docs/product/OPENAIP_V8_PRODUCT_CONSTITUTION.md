# OPENAIP_V8_PRODUCT_CONSTITUTION

各路 AI 工具都是英雄，OpenAIP 是指挥中心。

OpenAIP v8 定位为 AI Agent Control Plane / AI tool command center。
v7 是稳定基线，v8 是受控演进。
Agent Center 是主线；Task Center 目标是降低人类疲劳。
Provider Manager 吸收 CC Switch-like provider/config/router strengths。
OpenClaw 是可选但一等公民的 agent/runtime gateway integration。
OpenAxiom 定位为 Local App / UI Lab / Vision Tool，不作为 provider 主类。
Memory/Knowledge 是长期项目大脑。
Audit/Receipt 是强制证据系统。
Execution Gateway 默认 closed；Stage C 默认 disabled。
