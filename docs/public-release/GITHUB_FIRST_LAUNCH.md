## Release Notes
This is the first public launch of AegisFlow Intelligence Platform (AIP) Community Edition.
It includes Workflow Composer, Governance Hub, and a runnable local API/Web baseline.

### Included
- Workflow orchestration and run tracking
- Governance visibility surfaces
- Dataset/training/evaluation backbone routes
- Public-safe docs for architecture and security boundaries

### Known limits
- Private production artifacts are not included
- OpenClaw full production integration requires user-side configuration
- Local secrets and private datasets must not be committed

### Next
- Improve composer stability and observability
- Add more reusable community templates
- Strengthen contributor guides
