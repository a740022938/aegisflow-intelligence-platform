## 3. 数据模型

### knowledge_entries
| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT (PK) | UUID |
| title | TEXT | 标题 |
| category | TEXT | failure_postmortem / model_conclusion / task_experience / general_note |
| source_type | TEXT | task / model / experiment / general |
| source_id | TEXT | 关联实体 ID |
| summary | TEXT | 摘要 |
| problem | TEXT | 问题描述 |
| resolution | TEXT | 处理过程 |
| conclusion | TEXT | 结论 |
| recommendation | TEXT | 建议 |
| tags_json | TEXT | 标签数组 |
| created_at | TEXT | 创建时间 |
| updated_at | TEXT | 更新时间 |

### knowledge_links
| 字段 | 类型 | 说明 |
|------|------|------|
| id | TEXT (PK) | UUID |
| knowledge_id | TEXT | 知识条目 ID |
| target_type | TEXT | task / model / experiment |
| target_id | TEXT | 目标实体 ID |
| relation_type | TEXT | relates_to / blocks / resolves / improves |

---

## 4. API 设计

| 端点 | 方法 | 说明 |
|------|------|------|
| /api/knowledge | GET | 查询知识条目（支持 category / source_type / tag 过滤） |
| /api/knowledge | POST | 新增知识条目 |
| /api/knowledge/:id | GET | 知识条目详情 |
| /api/knowledge/:id | PUT | 更新知识条目 |
| /api/knowledge/:id | DELETE | 删除知识条目 |
| /api/knowledge/:id/links | GET | 知识条目关联的所有挂接 |
| /api/knowledge/link | POST | 建立知识关联 |
| /api/knowledge/unlink | POST | 解除知识关联 |
| /api/knowledge/by-entity/:type/:id | GET | 查询某实体关联的知识 |
