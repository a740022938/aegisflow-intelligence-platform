## 2. 安装/启停/升级流程

### 安装流程
```
1. 环境检查
   - Node.js 22.x: node --version
   - Python 3.11+: python --version
   - SQLite 3: sqlite3 --version

2. 代码获取
   - git clone <repo>
   - cd AGI_Factory/repo

3. 依赖安装
   - cd apps/local-api && npm install
   - cd apps/web-ui && npm install

4. 数据库初始化
   - 自动在首次启动时创建

5. 启动服务
   - API: cd apps/local-api && npx tsx src/index.ts
   - UI: cd apps/web-ui && npx vite --port 3000
```

### 启动步骤
```
# 启动 API server
cd E:\AGI_Factory\repo\apps\local-api
npx tsx src/index.ts
# → Running on http://0.0.0.0:8787

# 启动 Vite dev server
cd E:\AGI_Factory\repo\apps\web-ui
npx vite --port 3000
# → Running on http://localhost:3000
```

### 停止步骤
```
# 查找并关闭 API server
Get-NetTCPConnection -LocalPort 8787 | Select-Object -ExpandProperty OwningProcess | ForEach-Object { Stop-Process -Id $_ }

# 查找并关闭 Vite dev server
Get-NetTCPConnection -LocalPort 3000 | Select-Object -ExpandProperty OwningProcess | ForEach-Object { Stop-Process -Id $_ }
```

### 升级步骤
```
1. 停止服务（见上方）

2. 拉取最新代码
   git pull origin master

3. 更新依赖
   cd apps/local-api && npm install
   cd apps/web-ui && npm install

4. 重启服务（见上方）

5. 运行回归验证
   python scripts/regression_v500.py
```

## 3. 备份/恢复标准作业

### 备份产物清单
| 类型 | 文件名模式 | 内容 |
|------|-----------|------|
| DB Snapshot | `v{version}_db_snapshot_{timestamp}.db` | SQLite 数据库完整快照 |
| Seal ZIP | `v{version}_sealed_{timestamp}.zip` | 完整封板包（含 DB、Manifest、代码快照） |
| Seal Manifest | `v{version}_seal_manifest_{timestamp}.json` | 封板清单（版本、checksum、产物路径） |

### 备份操作步骤
```
1. 确保 API server 正在运行

2. 执行备份脚本
   node scripts/backup.mjs

3. 备份产物输出到:
   E:\AGI_Factory\backups\

4. 验证备份完整性
   - 检查 .db 文件大小 > 0
   - 检查 .zip 文件大小 > 0
   - 检查 manifest JSON 可解析
```

### 恢复前检查项
- [ ] 备份文件存在且完整
- [ ] 有足够的磁盘空间
- [ ] 当前服务已停止
- [ ] 已备份当前数据库（防止覆盖）

### 恢复操作步骤
```
1. 停止 API server

2. 备份当前数据库（如需要）
   copy packages\db\agi_factory.db packages\db\agi_factory.db.bak

3. 执行恢复脚本
   node scripts/restore.mjs --backup <backup_file.db>

4. 重启 API server

5. 运行恢复验证
   python scripts/recovery_verify.py --drill
```

### 恢复后验证项
- [ ] API health check 返回 ok
- [ ] Factory Status 页面可访问
- [ ] 最近发布记录可见
- [ ] Regression test 通过
