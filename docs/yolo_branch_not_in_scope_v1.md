# YOLO 支线暂不施工项 V1

## 1. 目的
明确本阶段明确不做的内容，避免误把“图纸轮”当“实现轮”。

## 2. 暂不施工项（明确禁止）
1. 不直接实现完整五连发全部生产级能力。
2. 不直接接入自动训练/自动评估/自动部署全链自动化。
3. 不重做 AGI Factory 主线架构。
4. 不把主线页面整体替换成支线页面。
5. 不做高风险数据库大迁移。
6. 不在本轮引入大量第三方依赖。
7. 不做未审批的线上发布策略。
8. 不将 OpenClaw 写成“已深度接入完成”。

## 3. 暂不施工原因
- 当前轮次目标是接力准备，不是重开发。
- 十连发涉及多模块耦合，必须分阶段降低风险。
- 主线已封板，任何大改都增加回退成本。

## 4. 后续触发条件（达到才可施工）
1. 接口契约稳定并通过样例验证。
2. 五连发基础闭环通过回归。
3. 审批门禁与审计模板准备完成。
4. 明确资源预算与失败回退方案。

## 5. 与主线关系声明
- 支线开发必须与主线隔离，避免污染 `v6.5.0` 封板基线。
- 主线历史债务（typecheck/workflow routes 等）不自动并入本支线。
