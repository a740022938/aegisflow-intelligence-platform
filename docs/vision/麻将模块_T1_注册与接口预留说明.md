## 文件属性

- **版本**: V1.0
- **阶段**: T1 注册与接口预留
- **日期**: 2026-04-15
- **依据**: 《麻将专用识别模块方案图纸_正式版_V1》《麻将专用识别模块施工任务包_正式版_V1》

## 1. 目标

在 AGI Model Factory 主骨架内注册麻将识别能力，挂接到 Tasks、Templates、Datasets、Training、Models、Audit 六大模块。仅做接口预留，不做全量硬接。

## 2. 已完成注册

### 2.1 Vision Bus 管线（vision-bus/index.ts）

| 管线 ID | 标签 | 状态 | 说明 |
|---------|------|------|------|
| mahjong_detect | 麻将检测 | planned | 复用 yolo_detect step_key |
| mahjong_classify | 麻将分类 | planned | 复用 classifier_verify step_key |
| mahjong_fusion | 麻将联合识别 | planned | yolo_detect + classifier_verify |

所有管线 status='planned'，不触发任何推理逻辑。

### 2.2 Artifact 类型注册

| 类型 | 描述 |
|------|------|
| mahjong_detection_result | 麻将检测结果（检测框 + 类别 + 置信度） |
| mahjong_classification_result | 麻将分类结果（34 类牌面判别 + 置信度） |
| mahjong_fusion_result | 麻将联合识别结果（检测 + 分类融合 + 最终置信度） |

### 2.3 Workflow 模板（workflow/index.ts SEED_TEMPLATES）

| 模板 ID | Code | Category | 状态 |
|---------|------|----------|------|
| tpl-mahjong-detect | mahjong_detect | mahjong_vision | planned |
| tpl-mahjong-classify | mahjong_classify | mahjong_vision | planned |
| tpl-mahjong-fusion | mahjong_fusion | mahjong_vision | planned |

### 2.4 各模块预留位

| 模块 | 预留方式 | 是否修改代码 |
|------|---------|------------|
| Tasks | 通过 template_code 关联模板 | 否 |
| Datasets | dataset_type='detection' + tags=['mahjong'] | 否 |
| Models | task_type='mahjong_detect', model_family='mahjong' | 否 |

## 3. 预留接口定义

### 3.1 VisionModuleInput（预留）

```typescript
interface VisionModuleInput {
  experiment_id: string;       // 关联实验 ID
  dataset_id: string;          // 数据集 ID
  model_id?: string;           // 模型 ID（可选）
  model_path?: string;         // 模型路径（可选，覆盖 model_id）
  execution_mode?: string;     // 执行模式：mahjong_detect | mahjong_classify | mahjong_fusion
  params?: Record<string, any>; // 额外参数
}
```

### 3.2 VisionModuleOutput（预留）

```typescript
interface VisionModuleOutput {
  pipeline_id: string;         // 管线 ID
  status: 'success' | 'failed' | 'partial';
  artifacts: Array<{
    type: string;              // artifact 类型
    path: string;              // 文件路径
    meta?: Record<string, any>; // 元数据
  }>;
  metrics?: Record<string, number>;  // 评估指标
  error?: string;              // 错误信息
}
```

### 3.3 兼容结构预留

| 角色 | 对应 step_key | 对应管线 ID | 状态 |
|------|--------------|------------|------|
| detector | yolo_detect | mahjong_detect | planned |
| segmenter | sam_segment | （SAM 已 active，麻将复用） | active |
| classifier | classifier_verify | mahjong_classify | planned |
| tracker | tracker_run | （通用，暂不接入麻将） | frozen |
| validator | rule_engine | （通用，暂不接入麻将） | frozen |
