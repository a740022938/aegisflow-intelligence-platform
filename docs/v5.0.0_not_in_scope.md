## 如发现少量必要改动

若实施中发现某些技术细节天然需要少量调整：

### 允许的微量调整
- 增加 DEFAULT 值的列
- 增加新的 API 端点（不影响现有）
- 增加新的 UI 页面（不重做现有）
- 增加新的脚本文件

### 必须遵守
- 必须是"增量式"
- 必须"兼容式"
- 必须"非破坏性"
- 必须汇报确认
