## 二、非阻断性问题（技术债）

### 高优先级

#### 1. 89个TypeScript类型错误
位置：apps/local-api/src/ 多个文件
影响：pnpm build exit code 2，不影响tsx运行时
分类：
- TS2339 (25个)：访问 {}.property，any类型滥用
- TS2345/TS2322 (23个)：SQLOutputValue类型不可赋值
- TS1470 (4个)：CJS输出禁用 import.meta
- TS2554 (5个)：函数参数数量不匹配
- TS2304 (3个)：packages变量作用域问题
- 其他 (29个)：ZodError.errors（Zod API变更）、crypto.v4缺失等
建议：逐文件修复，优先级：datasets > deployments > db
工作量：约2-3小时

#### 2. local-api build exit code 2
位置：apps/local-api/
影响：CI/CD pipeline可能无法使用pnpm build作为健康检查
建议：M6类型系统修复，或在CI中用 tsc --noEmit || true 绕过
