## 二、Schema 初始化方式

Schema 采用**自动建表**模式：在 `apps/local-api/src/db/builtin-sqlite.ts` 中，`getDatabase()` 函数首次被调用时自动执行：

1. 检查 .db 文件是否存在，不存在则创建
2. 逐表 `CREATE TABLE IF NOT EXISTS`
3. 逐列 `ALTER TABLE ADD COLUMN`（幂等，忽略已存在列）
4. 创建索引 `CREATE INDEX IF NOT EXISTS`
5. 注入默认 seed（如 route_policies 的 4 条默认策略）

**无需手动执行 init-db.js**。API 启动即自动完成全部 schema 初始化。

## 五、回滚说明

- 如需清空数据库：删除 `packages/db/agi_factory.db`，重启 API 即自动重建空库
- 如需恢复：从 Git 恢复或使用 `scripts/backup.mjs`
- M2 注入的种子数据为幂等 INSERT OR IGNORE，可安全重复执行
